# 🌐 ICT Project 106 Website

## 📌 Project Description
This project is an educational website created using Blogger (Blogspot platform).  
The website focuses on sharing knowledge related to Information and Communication Technology (ICT), including topics like internet usage, cybersecurity, digital tools, and basic computer concepts.

The website acts as a learning platform where users can read articles, understand ICT concepts, and explore different digital resources.  
A website is a collection of webpages containing multimedia information such as text, images, and links for users to access online resources. :contentReference[oaicite:0]{index=0}

## ⚙️ How to Run the Project

Since this project is hosted on Blogger:

1. Open the website link in any browser  
2. No installation required  
3. To edit:
   - Login to Blogger dashboard  
   - Go to "Theme" or "Posts"  
   - Modify content
